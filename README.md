## 📌 Overview
A comprehensive relational database architecture designed for a travel planning and services platform. This project models the complex relationships between users, bookings, travel destinations, and services, ensuring data integrity, scalability, and fast query performance.

## ✨ Features
* **Relational Modeling:** Well-structured tables with Primary and Foreign Keys ensuring ACID compliance.
* **Optimized Queries:** Pre-written SQL scripts for common operations (e.g., booking retrieval, user history).
* **Entity Relationships:** Covers customers, flight/bus tickets, hotel bookings, and service packages.

## 🛠️ Technologies Used
* **Database:** PostgreSQL
* **Management Tool:** pgAdmin
* **Language:** SQL (DDL, DML, DQL)
